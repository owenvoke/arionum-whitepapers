# arionum-whitepapers

[![Build Status][ico-travis]][link-travis]

A version-controlled backup of Arionum whitepapers.

## Install

Via Git

```bash
$ git clone git@github.com:pxgamer/arionum-whitepapers
```

## Usage

```bash
$ npm run build
```
